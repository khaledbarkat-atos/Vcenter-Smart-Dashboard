Recommendation Engine

0

Automatically scans all data sources and presents a single prioritised action list. Check items off as you handle them — your progress is saved. Click any card to navigate to the relevant page.

Storage vMotion Planner

Drag VMs between datastores to plan storage migrations. Usage bars update in real-time. Generate Storage vMotion PowerCLI script when ready.

Weekly Digest

Compares your current data against the oldest entry in History.json. Load history data on the Historical Trends page first.

Anomaly Detector

Uses statistical analysis (Z-score) to detect VMs and hosts behaving abnormally compared to their normal pattern. If History.json is loaded (Historical Trends page), it uses the multi-day baseline; otherwise it uses the 24-hour pattern. Each result shows which baseline was used.

Capacity Crystal Ball

Uses linear regression on historical data to predict when each resource will reach 85% and 100%. Load History.json on the Historical Trends page for accurate multi-day forecasting.

Peak Hours Analyzer

Analyzes the 24-hour usage pattern averaged across all hosts to identify peak and quiet hours. Use the quiet window to schedule backups, patching, and batch jobs to avoid contention.
